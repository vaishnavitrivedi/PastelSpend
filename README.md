# PastelSpend
PastelSpend is a fully interactive expense tracker and budget planner built with vanilla JavaScript, HTML5, and CSS3. It features custom SVG-based data visualizations, real-time budget alerts, multi-tab navigation, and a responsive glassmorphic UI — all without relying on external frameworks or charting libraries.
# 💰 PastelSpend

**PastelSpend** is a personal finance and budget-tracking web application designed with a soft, pastel-themed glassmorphic UI. Built entirely with vanilla JavaScript, it demonstrates core frontend engineering skills — from custom data visualization to state management — without relying on frameworks or third-party libraries.

## ✨ Features

- 📊 **Custom SVG Data Visualizations** — Dynamic bar charts, donut charts, and animated progress rings built from scratch
- 💸 **Full Transaction Management** — Add, edit, delete, search, and filter income/expense records
- 🎯 **Smart Budget Tracking** — Set spending limits with visual progress rings and automatic threshold alerts (80% warning, 100% exceeded)
- 📝 **Sticky Notes Gallery** — Attach reflections/notes to transactions, displayed as a searchable sticky-note wall
- 📤 **CSV Export** — Download transaction history for offline record-keeping
- 🌍 **Multi-Currency Support** — Switch between USD, INR, EUR, GBP, and JPY
- 📱 **Fully Responsive** — Adapts seamlessly across desktop, tablet, and mobile
- 💾 **Persistent Storage** — All data saved locally via localStorage, no backend required

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Custom design system with CSS variables, Flexbox/Grid, glassmorphism effects
- **Vanilla JavaScript (ES6+)** — DOM manipulation, state management, and dynamic SVG rendering (no frameworks or chart libraries used)

## 🎯 Why I Built This

I wanted to deepen my understanding of core web fundamentals by building complex UI features — like interactive charts and a full CRUD data flow — from scratch, without leaning on libraries like Chart.js or React. This project sharpened my skills in DOM manipulation, SVG rendering, and responsive design systems.
